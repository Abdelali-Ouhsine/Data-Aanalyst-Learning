# Data Analyst Bootcamp – Simplon

## Overview
This repository documents my journey through a **6-month Data Analyst bootcamp at Simplon**.  
It contains projects, exercises, and notes developed throughout the program, covering the full data analysis workflow from data collection to insights and visualization.

The goal of this repository is to track my progress, showcase practical skills, and serve as a reference for future data analysis work.

---

## Bootcamp Objectives
- Understand the role of a Data Analyst and data-driven decision making  
- Collect, clean, and prepare data for analysis  
- Analyze data to extract meaningful insights  
- Communicate results clearly using visualizations and reports  
- Work on real-world datasets and collaborative projects  

---

## Key Skills & Tools
### Technical Skills
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Statistical analysis  
- Data visualization  
- Dashboard creation  

### Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL  
- Excel / Google Sheets  
- Power BI  
- **Tableau Public**  
- Jupyter Notebooks
